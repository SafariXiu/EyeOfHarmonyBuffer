## 🌀 模组简介

**EyeOfHarmonyBuffer** 是一个为「数值即机制」而生的强化向模组。  
主打特点：**力大砖飞、大脑清空**（~~维神饼干~~），让你在高压机械与爆炸数值中获得前所未有的爽感。

本模组通过高度可配置的方式，对游戏中的各类机制进行**增强、删改、重构与扩展**，适合想要进一步压榨机械潜力、加速进度、或追求极限数值体验的玩家与服务器。

---

## ✨ 主要特性

本模组提供大量可配置、可自定义的功能，包括但不限于：

1. **机器增强**
   - 为原有机器提供大幅数值提升与机制强化  
   - 可通过配置精细调节增强幅度

2. **机制修改 / 删除**
   - 移除或调整部分影响体验的原版 / MOD 机制  
   - 用更简单粗暴的方式达到你想要的目标

3. **配方增强**
   - 强化现有配方效率  
   - 降低流程复杂度或资源压力

4. **新增单步配方**
   - 为复杂工艺链添加**一键到位**的单步配方  
   - 适合后期或服务器高负载环境使用

5. **多方块机器（豪堪风味）**
   - 搭配多方块结构，解锁更强力的机械能力  
   - 目前多方块相关功能部分基于他模多方块框架支持

6. **联动增强（私货专区）**
   - 对部分模组进行定制级强化，包括但不限于：
     - `Twist-Space-Technology-Mod`（如：闪电尖塔等）
     - `123tech` 中部分机器与机制
   - 若你有自己的模组想要增强，欢迎在 Issue 中提出需求！

## ⚙️ 配置与自定义

**核心思想：几乎所有数值与机制都可以配置。**

你可以在配置文件中找到：

- 机器性能倍率（速度、能耗、产出等）
- 各类机制开关与行为调整
- 配方倍率与替代单步配方
- 多方块结构与行为相关配置
- 各种 Mixin 功能的启用 / 禁用项

> **提示：**  
> 模组的详细功能说明、各项增强的具体参数与说明，请以 **MOD 配置文件** 中的注释与字段为准。

## 🧪 版本兼容性

- **1.3.X 分支**
  - **推荐版本：272**
  - 若关闭配置文件中的 `HIPMixin` 功能，**可能**可以在 **270** 版本下使用  
    - 但若因此产生兼容问题，**不提供修复支持**
- **1.2.X 分支**
  - 对应 **270** 版本
- **261 版本**
  - 请使用针对 261 的 **特殊构建版本**  
  - 此版本仅保留与「鸿蒙之眼」相关的功能
- **1.4.X 分支**
  - **现已支持 28X 版本！**

---

## 🔧 Mixin 功能说明

- 自 **1.3.15 版本起**：  
  - 所有 **Mixin 相关功能默认均为关闭**
  - 如需使用这些增强功能，请前往配置文件中**手动开启对应条目**
- 某些 Mixin 功能具有较强侵入性，请谨慎搭配其他深度修改类模组使用。
